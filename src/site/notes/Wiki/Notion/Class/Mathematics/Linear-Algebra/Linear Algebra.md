## 1. 核心概念
#### **1. 基本定义**

*   **线性方程 (Linear Equation)**
    *   **定义**: 一个包含变量 `x₁, x₂, ..., xₙ` 的方程，如果可以写成以下形式，则称为线性方程：
        `a₁x₁ + a₂x₂ + ⋯ + aₙxₙ = b`
    *   **组成部分**:
        *   `x₁, ..., xₙ` 是 **变量 (variables)**。
        *   `a₁, ..., aₙ` 是 **系数 (coefficients)**，通常是已知的实数或复数。
        *   `b` 是 **常数项 (constant term)**。

*   **线性方程组 (System of Linear Equations / Linear System)**
    *   **定义**: 由一个或多个包含相同变量 (`x₁, ..., xₙ`) 的线性方程组成的集合。

#### **2. 方程组的解**

*   **解 (Solution)**
    *   **定义**: 一组数值 `(s₁, s₂, ..., sₙ)`，当这组值分别代入变量 `x₁, x₂, ..., xₙ` 时，可以使方程组中的 **每一个** 方程都成立。

*   **解集 (Solution Set)**
    *   **定义**: 一个线性方程组所有可能解的集合。

*   **等价方程组 (Equivalent Systems)**
    *   **定义**: 如果两个线性方程组拥有完全相同的解集，那么它们是等价的。

#### **3. 解的数量与分类**

一个线性方程组的解有且仅有以下三种情况：

1.  **无解 (No Solution)**
2.  **唯一解 (Exactly One Solution)**
3.  **无穷多解 (Infinitely Many Solutions)**

根据解的存在性，可以将方程组分为两类：

*   **相容的 (Consistent)**
    *   **定义**: 方程组 **至少有一个解**。
    *   这包括了 **“唯一解”** 和 **“无穷多解”** 这两种情况。

*   **不相容的 (Inconsistent)**
    *   **定义**: 方程组 **没有任何解**。
    *   这对应了 **“无解”** 的情况。
#### **4. 矩阵的尺寸与表示 (Matrix Size and Notation)**

*   **矩阵尺寸 (Size of a Matrix)**:
    *   矩阵的尺寸由它的 **行数 (rows)** 和 **列数 (columns)** 决定。
    *   一个 `m × n` 矩阵是一个拥有 `m` 行和 `n` 列的矩形数字阵列 (其中 `m` 和 `n` 都是正整数)。
    *   **约定**: 行数在前，列数在后。

#### **4. 阶梯形矩阵 (Echelon Forms)**

*   **行等价 (Row Equivalent)**: 如果一个矩阵可以通过一系列初等行变换得到另一个矩阵，则称这两个矩阵是行等价的。

*   **阶梯形 (Echelon Form / Row Echelon Form)**:
    *   如果矩阵 `A` 与一个阶梯形矩阵 `U` 是行等价的，那么 `U` 被称为 `A` 的一个 **阶梯形**。
    *   一个矩阵可以有 **多个** 不同的阶梯形。

*   **简化阶梯形 (Reduced Echelon Form)**:
    *   这是阶梯形的一种特殊且更严格的形式。
    *   如果 `U` 是 `A` 的简化阶梯形，那么 `U` 被称为 `A` 的 **简化阶梯形**。
    *   对于任何一个矩阵 `A`，其简化阶梯形是 **唯一** 的。

#### **5. 主元 (Pivots)**

*   **主元位置 (Pivot Position)**:
    *   **定义**: 矩阵 `A` 中的一个位置，该位置对应于 `A` 的 **简化阶梯形** 中的 **首位 1 (leading 1)**。
    *   **关键点**: 主元位置是在 **原始矩阵 `A`** 中定义的，但需要通过其简化阶梯形来确定。

*   **主元列 (Pivot Column)**:
    *   **定义**: 原始矩阵 `A` 中 **包含一个主元位置** 的列。
